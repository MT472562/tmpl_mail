テンプレートメール作成ツール [?] 使い方

基本 テンプレート
左のテキストエリアに本文のテンプレートを書きます。{名前} のように { } で囲んだ部分が変数になり、データテーブルの列として自動追加されます。変数名は日本語・英数字どちらでもOKです。
基本 データ入力
テーブルの各列がテンプレートの変数に対応します。1行＝1通のメールです。「+ 行を追加」で行を増やし、「×」で削除します。
基本 プレビュー
◀ ▶ ボタンで各行の置換結果を確認できます。オレンジ色の部分が変数から置き換わった箇所です。グレーは値が未入力の変数です。
基本 プロジェクト保存
テンプレートとデータは「プロジェクト」としてブラウザに保存できます。画面を閉じてもデータは残ります。保存したプロジェクトは一覧から選択して読み込めます。
CSV CSV出力
現在のデータテーブルをCSVファイルとしてダウンロードします。Excelで開いて編集できます。
CSV CSV読込
CSVファイルを読み込んでテーブルに反映します。
CSVの作成ルール：
  • 1行目に見出し行（ヘッダー）を書きます。見出しはテンプレートの変数名と合わせてください。（例: 名前,日時,科目）
  • 2行目以降がデータです。（例: 山田,6/10 14:00,数学）
  • 値にカンマや改行を含む場合は "..." ダブルクオーテーションで囲みます。
  • 文字コードはUTF-8を推奨します。Excelで保存する場合は「CSV UTF-8」形式を選んでください。
Excel→CSVの手順：
  1. Excelの1行目に変数名（名前, 日時 など）を入力
  2. 2行目以降にデータを入力
  3. [ファイル] → [名前を付けて保存] → [CSV UTF-8 (カンマ区切り) (.csv)] で保存
  4. このツールの「CSV読込」でそのファイルを選択
CSV よくあるミス
  • ヘッダー名がテンプレートの { } 内の名前と一致しないと列が空になります
  • Excelで「CSV (カンマ区切り)」を選ぶと文字化けするので「CSV UTF-8」を選んでください
  • 余計な空行が入ると空データの行が作成されます
出力 一括出力
  • すべてコピー — 全行の置換結果をまとめてクリップボードにコピー
  • テキスト一括DL — 全行をテキストファイルとしてダウンロード
  • mbox形式DL — メールクライアントで読み込めるmbox形式で出力
